Projet Mangaverse Laravel

Mangaverse est une application web développée en PHP avec le framework Laravel. Elle propose un système complet d'authentification utilisateur et permet aux utilisateurs de parcourir, ajouter, modifier et supprimer des mangas. L'application intègre également des fonctionnalités interactives, telles que les commentaires et la notation des mangas. La base de données MySQL est utilisée pour stocker toutes les informations de manière sécurisée et organisée.
