Analyseur de code Sonarqube

Le projet vise à mettre en place l'analyseur de code statique Sonarqube sur la baie informatique de l'école. SonarQube est un outil open-source d’analyse statique de code qui permet d’évaluer la qualité d’un projet logiciel. Il aide à identifier les vulnérabilités de sécurité, les bugs, la duplication de code et les mauvaises pratiques, tout en fournissant des indicateurs sur la maintenabilité et la fiabilité du code.

J’ai déployé SonarQube Community Edition sur un serveur Debian afin d’analyser la qualité de mes projets en PHP. Pour cela, j’ai installé Java 17 ainsi que PostgreSQL, qui sert de base de données pour SonarQube. J’ai également installé SonarScanner, l’outil permettant de lancer des analyses de code à distance et j'ai récupéré des rapports détaillés sur la sécurité, la maintenabilité et la duplication du code depuis l'interface web associée.
